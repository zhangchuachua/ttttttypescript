> 看了上面的讲解，前往记住如何分辨 **父类型** 和 **子类型** : 子类型更具体，所以对于联合类型与 interface class 表现不一样，对于联合类型来说，类型越少越具体，例如 `number` 就应该是 `number | string` 的子类型, interface 与 class 中成员越多越具体。
